A healthy-eating app (recipes, meal plans, nutrition tracking). Clone and run locally:

how to use:
1. Clone the repo
   - git clone https://github.com/Alshaima12/healthyBite.git
   - cd healthyBite

2. Frontend (example with npm)
   - cd client
   - npm install
   - npm start
   - Open http://localhost:3000

3. Backend
   - cd server
     - npm install
     - npm run dev
